# 分类型 Prompt 变体

针对不同内容类型，时间点定位的 Prompt 有专门的变体设计，以适应各自的内容特性。

---

## 娱乐综艺类（entertainment）

### 核心原则：娱乐完整、节奏自然、效果保持

| 维度 | 说明 |
|------|------|
| **娱乐完整** | 确保每个片段包含完整的娱乐价值和观看体验，不破坏笑点和精彩瞬间 |
| **节奏自然** | 遵循娱乐内容的自然节奏，保持观看的流畅性和愉悦感 |
| **效果保持** | 维护搞笑效果、互动魅力和娱乐氛围的完整性 |

### start_time 确定方法

- **话题引入**: 主持人或嘉宾开始引入新话题的时刻
- **游戏开始**: 游戏规则介绍完毕，正式开始的时刻
- **互动启动**: 嘉宾间开始新一轮互动的时刻
- **表演开场**: 才艺表演或展示正式开始的时刻
- **情境转换**: 节目环节或氛围发生明显转换的时刻

### end_time 确定方法

- **话题结束**: 当前话题讨论完毕，准备转向下一个话题
- **游戏结束**: 游戏环节完成，结果揭晓或总结完毕
- **互动完成**: 一轮完整的互动或对话结束
- **表演结束**: 才艺表演或展示完整结束
- **笑点结束**: 搞笑片段的笑点完全释放完毕

### 娱乐综艺特殊考虑

| 考虑 | 说明 |
|------|------|
| **铺垫保留** | 保留必要的背景铺垫和氛围营造 |
| **笑点完整** | 确保笑点的完整设置和爆发过程 |
| **互动自然** | 保持嘉宾互动的自然引入和发展 |
| **效果延续** | 保留笑点后的反应和回味时间 |
| **观众反应** | 包含现场观众或嘉宾的反应时间 |

### 时长控制

| 类型 | 时长 |
|------|------|
| 轻松娱乐（日常搞笑、简单互动） | 3-5分钟 |
| 深度娱乐（有内涵的幽默、复杂游戏） | 5-7分钟 |
| 精彩表演（才艺展示、重要访谈） | 6-8分钟 |
| 精彩游戏（复杂游戏、多轮互动） | 可延长至10分钟 |
| 重要访谈（深度访谈、重要爆料） | 可延长至12分钟 |

### 必须完整保留的内容

1. **完整笑点**: 从铺垫到爆发到反应的完整过程
2. **游戏环节**: 规则介绍、进行过程、结果揭晓的完整流程
3. **互动对话**: 嘉宾间完整的对话和互动过程
4. **才艺表演**: 表演的完整过程和观众反应
5. **情感表达**: 感人或温馨时刻的完整情感流程

### 必须避免的错误

❌ 在笑点爆发的中间位置切断
❌ 在游戏进行到关键时刻时结束
❌ 在嘉宾情感表达到一半时切断
❌ 在互动最精彩的时候中断
❌ 忽略观众反应和现场氛围
❌ 破坏娱乐内容的自然节奏

## 商业财经类（business）

### 核心原则：精准、完整、商业逻辑清晰

| 维度 | 说明 |
|------|------|
| **精准定位** | 时间戳必须严格对应商业话题相关的核心讨论内容 |
| **完整性** | 时间范围必须完整覆盖一个商业话题的所有关键论证点，确保商业逻辑完整 |
| **自然边界** | 优先在商业观点转换、案例切换或逻辑段落边界处定位时间点 |

### start_time 确定方法

- 定位到讨论该商业话题的**第一句核心观点**的开始时间
- 忽略该观点之前的无关寒暄、免责声明或过渡语
- 如果有重要的背景铺垫或数据引用，应适当前移起始时间

### end_time 确定方法

- **必须**是覆盖该商业话题核心讨论的**最后一句话的结束时间戳**
- 确保商业分析、投资建议、风险提示等核心内容完整，不出现逻辑断层
- 如果话题末尾有重要的风险提示或免责声明，应包含在内

### 商业内容时长控制

| 类型 | 时长 |
|------|------|
| 一般商业话题 | 3-8分钟 |
| 复杂分析（多维度分析） | 可适当延长至10分钟 |
| 市场快讯、政策解读 | 可压缩至2-3分钟 |
| 深度分析（行业研究、公司分析） | 可延长至12分钟 |

### 商业内容特殊处理

| 类型 | 处理方式 |
|------|----------|
| **数据密集段落** | 包含大量财务数据、市场数据的段落应完整保留 |
| **案例分析** | 完整的商业案例分析应作为一个整体，避免拆分 |
| **投资建议** | 投资策略和风险提示应配套出现，不可分离 |
| **政策解读** | 政策内容与影响分析应保持完整性 |

### 需要避免的错误

- 在财务数据讲解中间断开
- 将投资建议与风险提示分离
- 在案例分析的关键论证点处截断
- 忽略重要的市场背景信息

## 演讲脱口秀类（speech）

### 核心原则：表达完整、情感连贯、效果保持

| 维度 | 说明 |
|------|------|
| **表达完整性** | 时间戳必须严格对应完整的表达单元和内容段落 |
| **情感连贯性** | 时间范围必须完整覆盖情感表达的起承转合，确保情感逻辑完整 |
| **自然边界** | 优先在话题转换、情感转换或表达节奏的自然边界处定位时间点 |

### start_time 确定方法

- 定位到该话题表达的**第一句核心内容**的开始时间
- 忽略该内容之前的无关过渡、停顿或背景音乐
- 如果有重要的铺垫、引入或氛围营造，应适当前移起始时间

### end_time 确定方法

- **必须**是覆盖该话题核心表达的**最后一句话的结束时间戳**
- 确保观点阐述、故事讲述、笑点呈现等核心内容完整，不出现表达断层
- 如果话题末尾有重要的观众反应、笑声或掌声，应包含在内

### 演讲脱口秀内容时长控制

| 类型 | 时长 |
|------|------|
| 一般话题 | 3-8分钟 |
| 深度演讲（深度思考或复杂论述） | 可适当延长至10分钟 |
| 精彩段子（独立完整的段子或笑点） | 可压缩至2-3分钟 |
| 重要演讲（核心主题演讲或重要观点阐述） | 可延长至12分钟 |

### 演讲脱口秀特殊处理

| 类型 | 处理方式 |
|------|----------|
| **完整段子** | 包含铺垫、发展、笑点的完整段子应作为一个整体 |
| **情感表达** | 重要的情感渲染和氛围营造应完整保留 |
| **观众互动** | 与观众的互动、问答、现场反应应配套出现 |
| **观点论述** | 核心观点的提出、论证、总结应完整包含 |

### 不同类型的时间定位策略

| 类型 | 定位策略 |
|------|----------|
| **主题演讲** | 按观点层次定位：核心观点-支撑论据-总结升华，保持论证逻辑的完整性 |
| **脱口秀表演** | 按段子单元定位：独立段子-连续笑点-主题串联，保持幽默效果的完整性 |
| **访谈对话** | 按话题讨论定位：问题提出-深度交流-观点总结，保持对话的自然流畅 |
| **经验分享** | 按经历阶段定位：背景介绍-过程描述-感悟总结，保持故事的完整性 |

### 需要避免的错误

- 在段子的笑点处断开
- 将观点阐述与论证过程分离
- 在情感表达的高潮处截断
- 忽略重要的观众反应或现场效果

## 类型选择建议

在实际应用中，可以根据视频的主要内容类型选择对应的 Prompt 变体：

| 视频类型 | 推荐 Prompt 变体 |
|----------|------------------|
| 综艺节目、娱乐节目 | 娱乐综艺版 |
| 财经分析、投资讲解 | 商业财经版 |
| 演讲、TED、脱口秀 | 演讲脱口秀版 |
| 知识科普、教育视频 | 通用版 |
| 日常Vlog、生活分享 | 通用版 |
